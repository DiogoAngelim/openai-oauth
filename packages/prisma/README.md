# Prisma Schemas

This folder contains the Prisma schema and migration files for the project. The main schema is located at `schema.prisma`.

- `schema.prisma`: Main Prisma schema for the project.
- `migrations/`: Contains all migration files generated by Prisma Migrate.
- `migrate_chat_history.sh`: Script for migrating chat history.
- `rollback.sh`: Script for rolling back migrations.

## Usage

- To generate the Prisma client:
  ```sh
  pnpm run prisma:generate --filter backend
  ```
- To run migrations:
  ```sh
  pnpm run migrate:deploy --filter backend
  ```
- To rollback migrations:
  ```sh
  pnpm run db:rollback --filter backend
  ```

See the main `README.md` for more details.
